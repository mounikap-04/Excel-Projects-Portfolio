#  Sales Performance Dashboard

---

##  Project Overview
This project presents a Sales Performance Dashboard built using Microsoft Excel.  
The dashboard analyzes sales data to track performance, measure targets, and generate business insights.

---

##  Objective
The objective of this project is to analyze sales data and evaluate performance using key metrics such as total sales, target achievement, and executive-wise performance.

---

##  Key Insights
- Total Sales Overview  
- Target Hit % Analysis  
- Away From Target % Tracking  
- Sales Executive Performance Comparison  
- City-wise Sales Distribution  

---

## Tools & Technologies Used
- Microsoft Excel  
- Pivot Tables  
- Percentage Calculations  
- Data Cleaning  
- Data Visualization Techniques
